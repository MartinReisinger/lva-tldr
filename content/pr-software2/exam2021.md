## Frage 3

Gegeben ist (langer) Text als ein `String`-Objekt. Es soll ermittelt werden, wie oft die Buchstaben `'a'` bis `'z'` vorkommen. Das Vorkommen wird in einem Array `int[] freq` mit 26 Elementen gezählt, also `freq[0]` enthält die Anzahl des Vorkommens für `'a'`. Es wird nicht zwischen Groß- und Kleinschreibung unterschieden. Alle anderen Zeichen werden ignoriert.

Folgende Methode berechnet die Häufigkeit des Vorkommens der Buchstaben im Text `text` im Bereich `from` bis `to` (exklusive) mit einer sequentiellen For-Schleife:

```java
private static int[] computeFreqSeq(String text, int from, int to) {
    int[] freq = new int[26];
    for (int i = from; i < to; i++) {
        char c = Character.toLowerCase(text.charAt(i));
        int idx = c - 'a';
        if (idx >= 0 && idx < 26) {
            freq[idx]++;
        }
    }
    return freq;
}
```

Da der Text lange ist, soll das Fork-Join-Framework mit `RecursiveTask`s eingesetzt werden. Folgend ist eine Klasse `CharFreqTask` begonnen. `CharFreqTask` hat ein Feld mit dem Text `text` und die beiden Positionen `from` und `to`, die den zu behandelnden Bereich des Textes bestimmen.

Implementieren Sie die Methode `compute`, die

- bei einem Bereich `[from, to)` mit weniger als `THRESHOLD` Elementen die Berechnung sequentiell mit der Methode `computeFreqSeq` durchführen soll;
- sonst den Bereich in zwei Teile teilen, für die beiden Teile zwei `CharFreqTask` erzeugen, diese zur Ausführung bringen, die Ergebnisse abrufen und die beiden Ergebnisse zusammenführen soll.

::example-block{title="Solution" collapsed-label="Reveal solution"}
```java
@Override
protected int[] compute() {
    if (to - from < THRESHOLD) {
        return computeFreqSeq(text, from, to);
    }

    int middle = from + (to - from) / 2;
    CharFreqTask left = new CharFreqTask(text, from, middle);
    CharFreqTask right = new CharFreqTask(text, middle, to);

    left.fork();
    int[] rightResult = right.compute();
    int[] leftResult = left.join();

    for (int i = 0; i < leftResult.length; i++) {
        leftResult[i] += rightResult[i];
    }
    return leftResult;
}
```
